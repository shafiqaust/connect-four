A Connect Four Multiplayer game

To run the server you need:

1. Install NodeJS (https://nodejs.org/) (latest version)
2. run the command: cd connect-four 
3. run the command: npm i
4. run the command: node server.js
5. Just go, on => http://localhost:5004
6. Enter a player name, the game will generate a link to join room for the second player. Click the link to join the game.( This feature is more or less is applicable for the bonus section.)
7. The blinking dark blue color means, it is a wrong turn for the current player.
8. In general, playing the game in two different browser windows is a basic multiplayer game.
